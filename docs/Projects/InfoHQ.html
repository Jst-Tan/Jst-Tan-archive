InfoHQ

InfoHQ is a software developed by JstMStudio. InfoHQ is a light-weight wiki-like software developed by JstMStudio.

InfoHQ is a software coded in HTML, CSS and JavaScript, with no server-side or serverless languages so it could be deploy to any web hosting available, almost all web hosting is supported.

InfoHQ is highly inspired by MediaWiki, however it is made with the intention of being able to be deploy to any web-hosting platform.
